📜 Project Methodology & Privacy

1. Methodology

The Political Pulse Election Analytics Dashboard utilizes a real-time data aggregation engine. User input is captured via a secure login gateway that validates Gmail domains to ensure data integrity.

  • Weighting: The AI Prediction algorithm evaluates vote distribution and statistical probability.
  • Visualization: Using Chart.js, the system renders dynamic bar graphs without page refreshes.

2. Data Privacy

In compliance with modern standards, this application follows a Privacy by Design approach. No data is transmitted externally; all records are stored in the browser's LocalStorage. Once a Gmail token is used, it is flagged to prevent duplicate voting, ensuring a fair and secure process.
